# AR Menu Demo

Simple demo website with:
- Welcome screen
- Opens device camera
- Shows mock 3D item
- Quantity selector & order button
- Kitchen screen that displays the order

**How to use:**
1. Open `index.html`
2. Click "Open Camera"
3. Show item & place order
4. See order in `kitchen.html`

Feel free to improve it by adding real AR SDK or backend!
